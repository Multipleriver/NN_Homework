## 文件树说明
对于每次作业我都会更新 Homework{x} 文件夹，每次作业我不写单独的 README，就当是合并在报告里面。
- `NN_Homework\Homework{x}`：每次作业单独一个文件夹
    - `NN_Homework\Homework{x}\project`：作业的项目主文件夹
        - `NN_Homework\Homework{x}\project\programs`：项目程序和代码，如`main.py`、`model.py`、`dataset.py`等
        - `NN_Homework\Homework{x}\project\output`：项目程序输出产物
        - `NN_Homework\Homework{x}\project\report`：项目报告和报告包含的内容
        - `NN_Homework\Homework{x}\project\resource`：项目所用到的资源文件
